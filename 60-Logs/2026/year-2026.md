# Annual Goals (SMART原則に基づく年間目標)

| 領域                  | SMART目標                                                                                                                                                   | 関連性                                                         | 年間達成度  |
| ------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------- | ------ |
| **修士研究**<br>(論文、学会) | **12/01**までに、**修士論文の文字数**を **0文字** から **30000文字** に改善することで、 **発信に対する影響力をつけること**を実現する。<br><br>そのために **修士論文の構成検討から執筆・図表の作成** を行う。                           | **自律的に生きる人を増やす** に対して **その実効性を担保する** ため                     | [ ] 0% |
| **就職活動**            | **4/1**までに、**NTTDATAか阪急阪神から入社先を選択** 、**入社先での知り合いの数** を**0人**から**10人**に改善することで、 **やりたいことを実現したい人を探すこと** を実現する。<br><br>そのために**入社先の決断と内定式・親睦会への最大限の参加** を行う。  | **やりたいことを実現したいと考えている人をサポートすること** に対して **それを求めている人を見つける** ため | [ ] 0% |
| **スキル開発**           | **10/01**までに、**データ分析の実務経験** を **RAKSULのみの2週間** から **2ヶ月の実務経験あり** に改善することで、**データドリブンな意思決定のサポート力向上** を実現する。<br><br>そのために **関西圏のベンチャーでの長期インターン** を行う。        | **日本でデータドリブンな意思決定を支えること** に対して **実行力を担保する** ため              | [ ] 0% |
| **健康・生活基盤**         | **12/01**までに、**ベンチプレスの挙上重量** を **80kg** から **100kg** に、**体脂肪率** を **21%** から **15%** に改善することで、**健康的に生きること** を実現する。<br><br>そのために **週3回の有酸素運動とベンチプレス** を行う。 | **mission statementの「体を大事にしよう」**と **健康維持の観点から関連する** ため      | [ ] 0% |
| **人間関係**            | **12/01**までに、 **指摘されたことを素直に受け入れる確率** を **40%** から**80%** に改善することで、 **全ての人に敬意を払い、個人が持つ意見は尊重しよう** を実現する。<br><br>そのために**[具体的行動]**を行う。                        | **mission statement** と **直接に関係する** ため                      | [ ] 0% |
| **趣味・財務**           | **6/1**までに、**家計簿の確認頻度** を **月1回** から **週1回** に改善し、その後も継続することで、**安定した家計管理** を実現する。<br><br>そのために**週1回、家計簿管理のための時間をカレンダーに登録すること** を行う。これはすべての予定より優先される。      | 長期ゴールの **金銭面で自由になる** に対して **自由になるための元手を増加させる** ため           | [ ] 0% |
