# Key Points

- 定期的な活動やプロジェクトの成果を評価し、継続的な改善を促すための「振り返り」フレームワークを体系的にまとめている。
- 各フレームワークには特有の目的と焦点があり（例：戦略評価、アジャイル改善、個人学習）、状況に応じて使い分けることが重要。
- 代表的な手法として、AAR、YWT、OKR振り返り、バランススコアカード、4Lメソッドなどがある。

# Context
- 定期的な振り返りは、個人や組織が経験から学び、現在地を正確に把握するために不可欠である。
- プロセスや戦略の有効性を客観的に検証することで、非効率な活動を削減し、より効果的な次のアクションを計画できる。
- チームで共通の認識を形成し、目標達成に向けたアライメントを高める効果も期待できる。

## 各フレームワークの概要

- **AAR (After Action Review)**: 「計画」「実績」「原因」「改善策」の4点で厳密に分析し、次に活かすための具体的な教訓を導き出す。
- **YWT**: 「やったこと」「わかったこと」「次にやること」の3点でシンプルに整理。個人の内省と次のアクション宣言を促す。
- **OKR振り返り**: 設定した目標(O)と主要な結果(KR)の達成度をスコアリングし、成功・失敗要因を分析して次のOKR設定に活かす。
- **バランススコアカード**: 企業の戦略を4つの視点から評価する経営管理手法。定期的なレビューにより戦略の妥当性を検証する。
- **4L**: ポジティブな点(Liked)、学び(Learned)、不足点(Lacked)、願望(Longed for)をチームで共有し、共感に基づいた改善アクションに繋げる。

## フレームワーク選択のガイドライン

適切な振り返りフレームワークを選択するには、以下の要素を考慮する必要がある。

### 1. 目的と文脈による選択

| 目的・状況 | 推奨フレームワーク | 理由 |
|------------|-------------------|------|
| **重大なプロジェクト失敗後** | AAR | 厳密な原因分析が必要。同じ失敗を繰り返さないための教訓抽出 |
| **日常的な個人の学習** | YWT | シンプルで継続しやすい。毎日の習慣化に適している |
| **2週間スプリントの終了時** | KPT、4L | 短サイクルに適した軽量な手法。チームの心理的安全性も維持 |
| **四半期の目標達成評価** | OKR振り返り | 目標とKRの達成度を定量的に評価し、次期戦略に反映 |
| **経営戦略の年次レビュー** | バランススコアカード | 多角的な視点で組織全体の健全性を評価 |
| **チーム形成初期** | 4L、YWT | 心理的安全性を重視。批判的になりすぎない |

### 2. チームの成熟度による選択

- **新しいチーム**: 4L、YWTなど、心理的ハードルが低い手法から始める
- **成熟したチーム**: AAR、OKRなど、より深い分析と挑戦的な目標設定が可能
- **混成チーム**: KPTなど、バランスの取れた手法で共通認識を形成

### 3. 時間的制約による選択

- **5-15分**: YWT（個人）、KPT（簡易版）
- **30-60分**: AAR、KPT、4L
- **2-4時間**: OKR振り返り、バランススコアカード

### 4. 複数フレームワークの組み合わせ

状況に応じて複数のフレームワークを組み合わせることも効果的。

**例：プロジェクトマネジメント**
- **週次**: KPTで短期的な改善点を抽出
- **月次**: AARで深い原因分析と学習
- **四半期**: OKR振り返りで目標達成度を評価
- **年次**: バランススコアカードで戦略全体を見直し

## 効果的な振り返りの実践ポイント

1. **定期的な実施**: スポット的ではなく、習慣として組み込む
2. **心理的安全性の確保**: 失敗を責めない文化を醸成
3. **具体的なアクション**: 「次にやること」を明確にし、実行可能にする
4. **記録と追跡**: 振り返りの内容を記録し、改善の進捗を追跡
5. **適切なファシリテーション**: 特定の人の意見に偏らないよう配慮

## Related Concepts

- **PDCAサイクル** - a continuous improvement method
    - ほとんどの振り返りフレームワークは、PDCAにおけるCheck（評価）とAct（改善）のフェーズを構造化し、実践するための具体的な手法と見なせる。
- **アジャイル開発** - an iterative development methodology
    - 4LやKPTなどの軽量な振り返り手法は、アジャイル開発におけるスプリントごとの「検査と適応」のサイクルで中心的な役割を果たす。
- **心理的安全性 (Psychological Safety)** - a shared belief that the team is safe for interpersonal risk-taking
    - 効果的な振り返りを実施するための土台となる。特に4LやYWTなどは、この安全性を高めながら進めることを重視する。
- **SMARTフレームワーク** [[20260102_smart-framework]] - goal setting framework
    - 振り返りの「次にやること」を具体化する際に、SMART原則を適用することで実行可能性が高まる。
- **プロジェクトマネジメント** [[ST_project-management]] - comprehensive project management framework
    - 振り返りフレームワークをPDCAサイクルの一部として体系的に活用するための全体像。目標設定から実行、評価、改善までを統合。

# 応用例

## スタートアップでの複合活用
**成長フェーズに応じた振り返り体系**
- **創業期（～10名）**: 週次YWT（個人）+ 月次AAR（全体）
  - 個人の学習速度を重視しつつ、重要な意思決定を深く振り返る
- **拡大期（～50名）**: スプリントKPT + 四半期OKR
  - チーム単位の改善サイクルと、組織全体の目標管理を両立
- **成長期（50名～）**: 部門別BSC + 全社OKR
  - 戦略の多角的評価と、部門間のアライメント確保

## 個人のキャリア開発
**マルチレイヤー振り返り**
- **毎日**: YWTで学習内容を定着（5分）
- **毎週**: KPTで業務改善点を抽出（15分）
- **四半期**: AARで大きなプロジェクトを深く分析（1-2時間）
- **年次**: 個人版BSCでキャリア全体を評価（スキル、人間関係、財務、健康など）

## 教育現場での活用
**学生の自律的学習支援**
- **授業後**: YWTで学びの定着（3分）
- **プロジェクト終了時**: AARでグループワークの振り返り（30分）
- **学期末**: 4Lで感情面も含めた総合的な振り返り（45分）
  - 「Longed for」で次学期への期待や目標を明確化
